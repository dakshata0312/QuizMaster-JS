# QuizMaster-JS
An interactive Quiz Application built with Vanilla JavaScript, featuring dynamic question rendering, real-time score tracking, and conditional state management for a seamless user experience.
## 🚀 Features
* **Dynamic Rendering:** Questions and choices are injected into the DOM from a centralized data array.
* **State Management:** Tracks user progress (current question) and total score across multiple rounds.
* **Interactive UI:** Features a start screen, real-time feedback after selecting an answer, and a final results summary.
* **Restart Capability:** Allows users to reset their progress and try again without refreshing the page.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3
* **Scripting:** JavaScript (ES6+)

## 📖 What I Learned
Through this project, I implemented several key software engineering concepts:
1. **Array Manipulation:** Navigating through objects within an array using a `currentQuestionIndex`.
2. **Conditional Logic:** Managing visibility of different UI containers (Start, Question, Results) using CSS classes.
3. **Event Driven Programming:** Using listeners to trigger complex functions like `startQuiz()` and `showResult()`.
4. **DOM Clearing:** Using `innerHTML = ""` to ensure a clean state between different question views.
